# 隐私说明

本项目不收集、不上传、不出售任何个人数据。

扩展和用户脚本只在以下页面运行：

- `http://202.115.194.60/Index.aspx*`
- `http://202.115.194.60/ExamManage/*`
- `http://202.115.194.60/SelfPrint/*`

处理 PDF 时，页面内容只在本地浏览器中用于生成 PDF。Chrome / Edge 扩展版会使用浏览器的 `Page.printToPDF` 接口，生成结果直接预览或下载到本机。
临时打印页会加载原页面引用的样式表、图片和水印等资源，以保持 PDF 排版一致。

项目没有远程统计、错误上报、广告、第三方 SDK 或外部服务。
